# AI Resume Analyzer

An AI-based web application that analyzes resumes against job descriptions and calculates an ATS compatibility score.

## Features
- Resume PDF upload
- Job description analysis
- ATS score calculation
- Skill match detection
- Missing skills identification
- Resume improvement suggestions
- Modern web interface

## Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python
- Flask

Machine Learning:
- TF-IDF Vectorization
- Cosine Similarity

## How it Works
1. Upload resume
2. Paste job description
3. System extracts resume text
4. AI compares resume and job description
5. Generates ATS score and suggestions
